タイトル：Almoth Training Plan - AIが導く、あなただけの進化の軌跡

概要： Almoth Training Planは、サイクリストやアスリートが自身のパフォーマンスを極限まで引き出すための、次世代型AIトレーニングダッシュボードです。Intervals.icuやHevyといった主要なプラットフォームと連携し、AIがあなたのデータを多角的に分析。単なるログ記録に留まらず、未来のパフォーマンスを予測し、最適なプランを提案します。

主な機能：

AIアスリートレベル推定： VO2max、FTP、過去のワークアウトデータから、AIが現在の「真の実力」を精密に評価。
ダイナミック・プラン生成： あなたの目標と現在のフィットネスレベルに合わせ、AIが個別化された週単位のトレーニングプランを作成します。
高度なメトリクス可視化： CTL / ATL / TSB理論に基づいたトレーニング負荷管理や、FTPの推移を洗練されたチャートで表示。
マルチデバイス同期： Vercel & Supabaseによるクラウド同期で、PCでの詳細分析からスマホでのクイックチェックまで、シームレスに利用可能。
モダンUXデザイン： グラスモーフィズムを採用した直感的で美しいUIが、日々のトレーニングへのモチベーションを高めます。
